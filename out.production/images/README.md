# Image Assets Directory

This directory contains all images used throughout the Bright Support website.

## Directory Structure

```
images/
├── hero/              - Homepage hero images
├── about/             - About Us page images  
├── services/          - Service-specific images
├── career/            - Career page images
├── contact/           - Contact page images
└── general/           - General purpose images
```

## Image Specifications

- **Format**: WebP (preferred) with JPG fallback
- **Optimization**: Images should be optimized for web (<200KB)
- **Resolution**: 72 DPI minimum (144 DPI for retina displays)
- **Color Space**: sRGB
- **Color Palette**: Indigo, Purple, Amber, Pink tones to match website theme
